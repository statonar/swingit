# SwingIT 1.0 — Decision Engine

A one-page Morning Playbook for high-conviction capital decisions.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Philosophy

SwingIT does not try to show every stock.
It prioritizes the few opportunities worth your attention today.

Core workflow:

1. Market pulse
2. Today's best opportunities
3. Expand investment thesis
4. Review evidence, risks, and trade plan
5. Decide whether to deploy capital
